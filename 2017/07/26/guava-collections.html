Google Guava 介绍2（集合）

发表于 2017-07-26 | 分类于 JAVA | 阅读次数

Guava项目包含我们在基于Java的项目中依赖的几种Google核心库：集合，缓存，原始支持，并发库，常见注释，字符串处理，I / O等。这些工具中的每一个都真的被Google员工在生产服务中每天使用。

不可变集合

不可变对象有相应多的优点，其中包含：

  • 线程安全，不存在多线程竞争及锁的问题。
  • 不支持改变，在空间和时间上比较有优势
  • 可以常量使用
    JKD也提供Collections.unmodifiableXXX方法，使用不方便，并且不安全。

新集合

Guava扩展了多种JDK没有提供的集合类型，新扩展的集合类型都支持JDK集合规范

Multiset

在集合中元素被允许出现不止一次的出现。.在多重集合中，与集合相反，与元组相反，元素的顺序是无关的：多重集合{a，a，b}和{ a，b，a}相等。

  • 常用方法
方法 说明
count(E) 返回元素的数量，元素不存在返回0
elementSet() 返回唯一的元素列表
entrySet() 返回Set>
add(E,count) 增加元素数量
remove(E,count) 减少元素数,如果数量小于等于0则删除元素
setCount(E,count) 设置元素数量 ，如果小于等于0则删除元素
size() 返回所有元素数量和
  • Multiset和Map不相同的地方
    • Map中可以出现数量小于0的情况
    • Multiset操作比较方便
  • 常用实现及JDK原生类
Map Corresponding Multiset Supports null elements
HashMap HashMultiset Yes
TreeMap TreeMultiset Yes (if the comparator does)
LinkedHashMap LinkedHashMultiset Yes
ConcurrentHashMap ConcurrentHashMultiset No
ImmutableMap ImmutableMultiset No
